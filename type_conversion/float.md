# Float to Int

## Descrição

Converter um float para um inteiro.

## Input

Um float.

## Output

Um Int.

## Exemplos

```python
num = 214.17

# your code here

""" expected output:
214
"""
```

<details>
    <summary>Resposta</summary>
<p>

```python
num = 214.17

convertedNum = int(num)

print(convertedNum)
```

</p>
</details>
